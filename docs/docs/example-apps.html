Example Applications

Two example applications and their complete source code are provided with the NixUniversalSDK download. To review the examples, use Visual Studio 2022 to open the nix-universal-sdk-windows-example.sln solution file located in the example-windows folder. The solution contains two projects, one for each example application.

Both applications demonstrate:

  • Device discovery on Bluetooth and USB interfaces
  • Opening and closing a connection to a device
  • Displaying additional device info (serial number, battery level, etc.)
  • Toggling device options (haptic feedback, RGB feedback, etc.)
  • Taking measurements
  • Obtaining colorimetry and spectral data from measurements
  • Performing in-field calibration on the provided reference tile
  • Displaying additional measurement information (temperature, scan settings, etc.)

Console Demo

The NixExampleConsole project contains a basic console application, targeting .NET Framework 4.7.2. To run this example, set NixExampleConsole as the 'Startup Project', build, and run.

WinUI3 Demo

The NixExampleWinUI project contains a WinUI3 based application, targeting .NET 7.0. To run this example, set NixExampleWinUI as the 'Startup Project', build, and run.
